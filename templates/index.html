PlotPalette

An application for tackling writer's block using generative AI and storyboarding.

Application Features

Storyboard

You can add, delete, and rearrange scenes. Click on the scene title and scene description to edit the text. You can upload an image to replace the placeholder image provided with each scene card.

You can also generate plot ideas and directly add the plot idea to your storyboard as a scene card. This scene card will be editable so you can modify the AI generated ideas if needed.

If you're stuck, try structuring your story using the Five Act Structure or the Hero's Journey by clicking the corresponding buttons to add these templates to your storyboard.

AI Image Generation

You can generate images of multiple scenes from your story featuring the same character or generate images of any settings, objects, creatures, etc. which are part of your narrative. To generate consistent images of a character, you must first generate a character description that the AI will reuse to generate subsequent images.
